Changes\n\n**CSS Changes**:\n1. Maintain semantic structure\n3. Keep canvas element\n4. Keep script imports at bottom\n5. Test navigation and reveal dot\n\n**JavaScript Changes**:\n1. Edit appropriate `js/*.js` file\n2. Test functionality\n3. Check browser console for errors\n4. Verify animations work\n\n### Testing Workflow\n\n```bash\n# Open in browser\nopen index.html\n\n# Or with Python server\npython -m http.server 8000\n# Then visit: http://localhost:8000\n```\n\n**Test Checklist**:\n- [ ] Navigation works\n- [ ] Theme toggle works\n- [ ] Mobile menu opens/closes\n- [ ] Reveal dot on first visit\n- [ ] Animations play smoothly\n- [ ] Responsive at 900px breakpoint\n- [ ] Links navigate correctly\n- [ ] Canvas background renders\n- [ ] Dark mode colors correct\n- [ ] No console errors\n\n---\n\n## 📊 CSS Organization\n\n### Variable System\n```css\n/* Light mode */\n:root {\n  --accent: #3b82f6;\n  --accent-light: #60a5fa;\n  --accent-dark: #2563eb;\n  --text: #1f2937;\n  --muted: #6b7280;\n  --border: #e5e7eb;\n  --panel: #ffffff;\n  /* ... more variables */\n}\n\n/* Dark mode */\nbody:not(.light) {\n  --text: #f3f4f6;\n  --muted: #9ca3af;\n  --border: #374151;\n  --panel: #1f2937;\n  /* ... dark mode overrides */\n}\n```\n\n### Section Comments\nEach major CSS section is marked with:\n```css\n/* =========================\n   SECTION NAME\n   ========================= */\n```\n\nThis makes navigation and organization easy.\n\n---\n\n## 🎨 Customization Guide\n\n### Change Colors\n1. Modify font import in `index.html` (head section)\n2. Update font sizes in `:root`\n3. Adjust line-heights in specific sections\n4. Test responsiveness\n\n### Change Animations\n1. Find `@keyframes` section in `css/style.css`\n2. Modify animation properties\n3. Adjust duration/easing in components\n4. Test animation performance\n\n### Change Spacing\n1.
